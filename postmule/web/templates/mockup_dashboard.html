Branding Audit — Gaps vs. Current app.py

GAP 1 — No logo SVG in header. The header renders text-only "PostMule". The brand spec requires the SVG mule mark (ears, eyes, muzzle, nostrils) alongside the wordmark. See header above for the correct version.
GAP 2 — No tagline in header. "Smart mail management" (#5A7CA4, 11px, uppercase, letter-spacing 2px) is missing beneath the wordmark. Brand spec defines this as a required element.
GAP 3 — Page title font-size is 22px (spec: 20px) and missing letter-spacing. All <h1> page titles use font-size:22px. Brand spec: 20px, weight 600, #0F2044, tracking -0.3px. Also missing: letter-spacing: -0.3px on all page title elements.
GAP 4 — Mail card badges are all the same blue style regardless of type. Current app.py renders every category badge as EEF3F9/2C4A6E (Notice blue). Brand spec requires: Bill = brass (#FDF3E0/#7A4C08), Overdue = red (#FEECEC/#8A1010), ForwardToMe = urgent red, Notice = steel blue, Junk = smoke, NeedsReview = amber.
GAP 5 — Read state not applied to mail card sender names. Brand spec: unread = weight 600, #0F2044. Read/Junk = weight 400, #8A9BB0. Current app hardcodes font-weight:600;color:#0F2044 for all items.
GAP 6 — Mail card body text is 12px (spec: 13px). The summary line under the sender uses font-size:12px. Brand spec body text is 13px, #5A7090.
What IS correct: All color tokens match (#0F2044, #E8A020, #7A9CC4, #F5F6F8, #DDE3EC, #C62828, #2E7D32). Header/nav background, page background, card structure (8px radius, 1px #DDE3EC border, 3px left bar), stat labels, toast pattern, table structure — all on-brand.

Rules
Only animate transform and opacity. Never animate height, width, or layout properties.

Icons

Always render icons at the same stroke weight. Do not mix icon sets. Size icons at 16px for inline/badge use, 20px for toolbar/action use. Color icons using the text color token of the surface they sit on — never use a one-off color for an icon alone.

Rules
Lead with the fact, not the framing. State the amount and due date first.
Use sentence fragments when they're clearer than full sentences.
No exclamation marks. No hedging language ("might", "seems like").
Precise numbers — always include cents when non-zero ($94.00, not $94).
Time references: "16 days remaining", not "soon" or "upcoming".
Errors are facts, not apologies: "Failed to fetch. Check credentials." not "Sorry, something went wrong."

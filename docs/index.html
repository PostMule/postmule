PostMule
Smart mail management

Your bills, handled automatically.

PostMule is an open-source, self-hosted bill manager. It processes your physical mail and email bills, AI-classifies them, extracts the data, and reconciles with your bank — every night while you sleep.

How it works

Set it up once. It runs every night.

PostMule wakes up on a schedule (default 2 AM), pulls your mail, processes every PDF, and updates your records — no manual steps required.

Step 1
Collect mail
Fetches physical mail scans from your Virtual Post Mail account and PDF attachments from your email inbox.
Step 2
AI classification
OCR extracts text, then a configurable LLM (default: Gemini 1.5 Flash, free tier) classifies each document and extracts key fields.
Step 3
File & record
Documents are renamed with a standard scheme, moved to cloud storage, and recorded in JSON files with a Google Sheets view generated automatically.
Step 4
Reconcile & alert
Bills are matched against bank transactions (YNAB, etc.), upcoming due dates trigger proactive alerts, and a daily summary email lands in your inbox.
Features

Everything a paper-based bill manager needs.

OCR + AI classification
pdfplumber extracts text; pytesseract handles image-only PDFs. Any LLM provider you configure classifies the document type and extracts fields.
Scheduled automation
Runs on Windows Task Scheduler at a configurable time. No server, no Docker — just Python running on your own machine.
Bank reconciliation
Pulls transactions from your finance provider (YNAB, etc.) and matches them to processed bills by amount and date.
Smart alerts
Immediate email when a "Forward To Me" item arrives. Proactive bill-due alerts. Daily summary delivered every morning.
Provider/adapter pattern
Every external service — mail, email, storage, LLM, finance — sits behind a swappable interface. Change providers with one config line.
Local web dashboard
Flask + HTMX + Alpine.js dashboard at localhost:5000. Review mail, manage entities, configure providers, and browse logs.
Get started

Install in minutes.

PostMule runs on Windows 11 with Python 3.12+. You'll need a Google account for cloud storage and a VPM or Gmail account to feed it mail.

1
Download the installer — grab PostMuleSetup.exe from the Releases page and run it. Python and dependencies are bundled.
2
Configure — edit config.yaml (created on first run) with your mail provider, storage, LLM key, and notification email.
3
Run the dashboard — open a terminal and run postmule dashboard. Visit http://localhost:5000 to review and approve the initial setup.
4
Schedule — run postmule schedule to register the nightly Task Scheduler job. PostMule wakes up automatically from then on.
Prefer to install from source? See the full installation guide in the repo. Detailed provider setup (Gmail OAuth, Google Drive, YNAB) is covered there.
